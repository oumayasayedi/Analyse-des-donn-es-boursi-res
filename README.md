# Analyse-des-données-boursières
Pipeline BigData pour l’alimentation de base de données en temps réel et analyse de données boursières.
L’utilisation conjointe de Kafka, Spark Streaming, Elasticsearch et Kibana, orchestrée par Airflow et déployée via Docker, a permis de garantir une architecture modulaire, scalable et automatisée.
Cette approche illustre l’efficacité des technologies Big Data modernes pour l’analyse de données financières et constitue une base solide pour des extensions futures telles que l’analyse prédictive ou la détection d’anomalies.
